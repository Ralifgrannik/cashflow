# Cashflow (ДДС)

Веб-приложение для учета движения денежных средств.

## Запуск проекта из GitHub

1. Клонируйте репозиторий:

```powershell
git clone https://github.com/<ваш-пользователь>/<имя-репозитория>.git
cd <имя-репозитория>
```

2. Создайте виртуальное окружение и активируйте его:

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

3. Установите зависимости:

```powershell
pip install -r requirements.txt
```

4. Выполните миграции базы данных:

```powershell
python manage.py migrate
```

5. Запустите веб-сервер:

```powershell
python manage.py runserver
```

6. Откройте приложение в браузере:

```text
http://127.0.0.1:8000/
```

## Примечания

- Проект использует SQLite, поэтому дополнительных настроек базы данных не требуется.
- В приложении есть начальные справочники для статусов, типов, категорий и подкатегорий.
- Если хотите использовать админ-панель Django, создайте суперпользователя:

```powershell
python manage.py createsuperuser
```

## Что реализовано

- Главная страница с таблицей записей ДДС и фильтрами по:
  - дате (период),
  - статусу,
  - типу,
  - категории,
  - подкатегории.
- Создание, редактирование и удаление записей.
- Удобная работа со справочниками:
  - статусы,
  - типы,
  - категории,
  - подкатегории.
- Логические зависимости:
  - категория привязана к выбранному типу;
  - подкатегория привязана к выбранной категории.
- Формат отображения суммы в рублях: `1 000 р.`.

## Структура проекта

- `dds/` — основное приложение Django.
- `dds/models.py` — модели для записей, статусов, типов, категорий и подкатегорий.
- `dds/forms.py` — формы создания/редактирования записи и справочников.
- `dds/views.py` — представления со всеми CRUD-операциями.
- `dds/templates/` — HTML-шаблоны интерфейса.
- `static/dds/dynamic_selects.js` — JavaScript для зависимых выпадающих списков.
